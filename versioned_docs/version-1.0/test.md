# 测试

## 模拟交易测试

:::info
完整体验需要开启VPN，否则选镜片功能加载不出
:::

|  字段名称  | 要输入的详细信息                                                                                                                                             |
| :--------: | :----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 持卡人姓名 | 依次输入以下各项内容来模拟不同类型的交易：`输入1 ，模拟成功的交易`；`输入2 ，模拟失败的交易`；`输入3 ，模拟异常情况（这会生成一条消息，指示出现提供商错误）` |
|    CVV     | 输入任意 3 位数字                                                                                                                                            |
|   有效期   | 输入一个将来的日期                                                                                                                                           |

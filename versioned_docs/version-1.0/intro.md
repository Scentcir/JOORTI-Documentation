# 概况

## 编写目的

JOO’RTI是基于Shopify平台搭建的出海电商网站，其目的是为了高效的布局跨境电商，快速与目标消费者建立联系。因此该项目会在某些方面存在诸多限制，此文档的目的便是清晰、有层次的定义页面原型中的各个模块的内容来源和相关的逻辑。

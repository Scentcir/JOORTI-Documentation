## Shopify 基础方案

### 基础方案费率

|                                      | 基础 Shopify     | Shopify          | 高级 Shopify     |
| :----------------------------------- | :--------------- | :--------------- | :--------------- |
| 价格                                 | US$29.00         | US$79.00         | US$299.00        |
| 网上香港信用卡费率                   | 3.3% + 2.35 港元 | 3.2% + 2.35 港元 | 3.1% + 2.35 港元 |
| 在线国际/美国运通信用卡费率          | 3.9% + 2.35 港元 | 3.8% + 2.35 港元 | 3.7% + 2.35 港元 |
| 不使用 Shopify Payments 时的交易费用 | 2.0%             | 1.0%             | 0.5%             |

使用 Shopify Payments 交易，无需额外手续费。如果您选择使用外部支付网关，Shopify 基础版用户则需额外支付 2%手续费，Shopify 进阶版用户支付 1%手续费，Shopify 高级版用户支付 0.5% 手续费，以此类推。

Shopify 除了每月的套餐费用外，Shopify 还会向我们这些用户收取其计划的 0.5％-2％的交易费用，除非您使用 Shopify Payments 为您的收款方式，在这种情况下, 你所有交易费用都将被免除。

### 不使用 Shopify Payments 时的交易费用

| 基础 Shopify | 29+3000\*2%=29+60=US$89        |
| :----------- | :----------------------------- |
| Shopify      | 79+10000\*1%=79+100=US$179     |
| 高级 Shopify | 179+50000\*0.5%=179+250=US$429 |

* **信用卡利率：**由第三方平台收取的费用
